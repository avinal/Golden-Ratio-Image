# Golden-Ratio-Image
Converts a image to a grayscale/color Fibonacci Number based Golden Ratio Image. 

*Use Squared-Cropped Image for best Results.*

# How to Compile
```
git clone https://github.com/avinal/Golden-Ratio-Image.git
cd Golden-Ratio-Image
javac src/*.java
java src.TreeDraw
```
